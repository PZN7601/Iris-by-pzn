# Iris

Iris is a cross-platform terminal AI agent designed to enhance your productivity by integrating seamlessly with Kali Linux. With its robust features, Iris provides a user-friendly interface for executing commands, managing system tasks, and automating workflows. Whether you're a security professional or a casual user, Iris aims to streamline your command-line experience and improve your overall efficiency.

## Features
- **Cross-Platform Compatibility**: Works on multiple operating systems.
- **Kali Integration**: Leverages tools from Kali Linux for security assessments.
- **User-Friendly Interface**: Simplifies interactions in the terminal.
- **Automation**: Automate repetitive tasks with ease.

## Getting Started
To get started with Iris, clone the repository and run the setup script.
